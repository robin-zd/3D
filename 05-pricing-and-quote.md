# 报价体系和客户报价模板

## 三档产品方案

### 方案A：整件白膜 + 颜料包

适合：
- 预算有限
- 想自己体验上色
- 结构简单的小摆件

包含：
- AI概念图
- 基础3D模型
- 整件白膜打印
- 简单打磨
- 颜料建议或基础颜料包

不包含：
- 精细拆件
- 成品上色
- 高复杂展示盒

### 方案B：拆件白膜 + 颜料包

适合：
- 想要更好的打印质量
- 模型结构比较复杂
- 计划自己上色但希望部件更清晰

包含：
- AI概念图
- 多视角参考图
- 3D模型生成和基础修整
- 拆件白膜
- 简单连接结构
- 颜料建议或颜料包

### 方案C：上色成品 + 展示盒

适合：
- 送礼
- 展示收藏
- 不想自己处理后期

包含：
- AI概念图
- 四视图
- 3D模型生成和修整
- 拆件/整件打印
- 打磨、上色、组装
- 基础展示盒
- 包装发货

## 内部报价公式

```text
总价 =
设计整理费
+ AI概念图费用
+ 建模/修模费用
+ 打印耗材费用
+ 打印机时费用
+ 拆件复杂度费用
+ 打磨费用
+ 上色费用
+ 展示盒费用
+ 包装运费
+ 加急费用
+ 风险缓冲
```

## 成本评估字段

```text
预计高度
预计耗材克重
预计打印时长
是否需要支撑
是否需要拆件
拆件数量
是否需要磁吸/插销/胶合
颜色数量
上色复杂度：低/中/高
展示盒复杂度：无/基础/定制
交付周期
风险点
```

## 客户报价模板

```text
根据你的需求，我建议给你3个方案：

A. 整件白膜 + 颜料包
适合自己动手上色，价格更低。
预估价格：【价格】
预估周期：【周期】

B. 拆件白膜 + 颜料包
打印质量和细节会更好，也更适合后续上色。
预估价格：【价格】
预估周期：【周期】

C. 上色成品 + 展示盒
我这边完成打印、打磨、上色和组装，收到后可以直接摆放或送人。
预估价格：【价格】
预估周期：【周期】

如果你是第一次定制，我更推荐【推荐方案】，因为它在预算和效果之间比较稳。
确认方向后，定金为【金额】，开始进入概念图和建模排期。
```

## 定金规则模板

```text
下单需要先付定金【金额/比例】，用于锁定排期和开始设计制作。

定金包含：
1. 需求整理
2. 概念图方向
3. 基础建模准备
4. 排期占位

确认进入制作后，如客户临时取消，定金不退。
如果是我这边判断无法制作或无法按约定交付，会退还定金。
```

## 修改规则模板

```text
概念阶段可免费调整【次数】次。
进入3D模型和打印阶段后，大结构修改需要重新评估费用和周期。
已开始打印后，不再接受造型大改。
```
