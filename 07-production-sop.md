## 模型检查清单

打印前必须检查：

```text
是否封闭/watertight
是否有非流形结构
最薄处厚度是否足够
是否有过细的头发、手指、配件
是否能稳定站立
是否需要底座
是否需要拆件
拆件连接方式是否明确
是否有无法上色的死角
是否有大面积悬垂
是否需要支撑
是否适合当前打印机尺寸
```

## 拆件判断

建议拆件的情况：

```text
多颜色且分色边界清晰
手臂、翅膀、武器等悬空明显
主体太大，单次打印风险高
上色遮挡严重
局部细节需要更好打印方向
需要降低支撑痕迹
```

不建议过度拆件的情况：

```text
客户预算低
模型非常小
连接位不容易隐藏
拆件后组装强度差
客户选择的是整件白膜
```

## Bambu Studio 半自动流程

Bambu Studio 支持命令行导入、自动朝向、自动摆放、切片和导出3MF。实际生产建议半自动，不建议完全无人值守。

示例命令：

```bash
/Applications/BambuStudio.app/Contents/MacOS/BambuStudio --orient --arrange 1 --slice 0 --export-3mf output.3mf model.stl
```

人工必须检查：

```text
打印方向
支撑位置
接触面
细节是否被支撑破坏
预估耗材
预估时间
层高是否合适
填充和壁厚是否合适
是否需要分盘打印
```

## 品控清单

发货前检查：

```text
主体是否完整
明显层纹是否可接受
支撑痕迹是否处理
尖锐毛边是否处理
连接处是否牢固
上色是否干透
展示盒是否能正常放入
包装是否防压防晃
是否拍摄发货前照片
```
