# 社媒获客手册

## 内容目标

前期内容的目标不是单纯展示技术，而是让用户相信：

```text
我可以把你的模糊想法，做成一个能收到、能摆出来、能送人的实体作品。
```

## 内容比例

```text
40% 成品案例：客户需求 → 成品
25% 制作过程：AI图 → 四视图 → 3D模型 → 打印 → 上色
20% 种草选题：生日、宠物、情侣、原创IP、桌面摆件
15% 避坑科普：定制流程、报价逻辑、AI图不能直接打印等
```

## 小红书九宫格结构

```text
第1张：成品最吸引人的照片或渲染图
第2张：客户需求/灵感一句话
第3张：AI概念图
第4张：前后左右四视图
第5张：3D模型截图
第6张：拆件图或打印摆盘
第7张：打印过程
第8张：上色过程
第9张：成品 + 展示盒 + 引导语
```

## 短视频结构

```text
0-2秒：客户只给我一句话，我把它做成了实体摆件
2-6秒：展示客户需求/参考图
6-12秒：AI概念图和四视图快速切换
12-20秒：3D模型、拆件、打印
20-28秒：上色、组装、展示盒
28-35秒：成品旋转展示 + 评论关键词
```

## 每周发布节奏

```text
周一：案例展示
周二：制作过程
周三：避坑科普
周四：样板项目/新风格测试
周五：成品案例或客户返图
```

- 这不是买的，是按她的想法定制出来的

- 为什么你的AI图不能直接拿去3D打印？

## 小红书正文模板：案例型

```text
这次的需求很简单，也挺有画面感：
客户想把【客户需求】做成一个可以摆在桌面上的小作品。

我先把需求整理成几个关键词：
【关键词1】/【关键词2】/【关键词3】

第一步先用AI做概念效果图，确认整体气质。
确认后再做前后左右四视图，用来辅助生成和修整3D模型。

后面会进入打印、拆件、上色和展示盒制作。
这类定制比较适合做生日礼物、宠物纪念、情侣纪念和原创IP小摆件。

想了解定制流程可以评论【定制】，我发你案例和需求表。

#3D打印 #原创定制 #手办定制 #小红书手作 #生日礼物 #宠物纪念 #桌面摆件
```

## 小红书正文模板：避坑型

```text
很多人以为有一张AI图，就能直接3D打印。
其实中间还差很多步。

AI图主要解决的是“长什么样”，但3D打印还要解决：
1. 能不能站稳
2. 厚度够不够
3. 哪些地方要拆件
4. 哪里需要支撑
5. 上色能不能做到预期效果

所以我的流程一般是：
需求沟通 → AI概念图 → 四视图 → 3D模型 → 打印检查 → 拆件/上色 → 成品交付。

想做定制的话，不用一开始就想得很完整。
你可以先发一句想法或几张参考图，我会帮你整理成可制作方案。
```
