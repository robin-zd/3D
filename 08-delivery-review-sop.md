# 发货、售后和好评引导

## 发货前通知

```text
你的定制作品已经制作完成，我这边会先拍一组发货前照片给你确认。

确认没问题后会进行包装发货。
尾款为：【金额】
收到尾款后安排寄出。
```

## 发货消息

```text
你的定制作品已经发出啦。

快递公司：【快递】
单号：【单号】

收到后建议先检查外包装和模型细节。
如果运输过程中有损坏，请第一时间拍照发我，我会帮你处理。
```

## 收货后关怀

```text
你那边收到后感觉怎么样？
如果方便的话，可以拍一张摆放图给我看看，我也想知道它最后被放在了哪里。
```

## 好评引导

```text
如果你喜欢这次定制，可以在小红书发一条返图笔记。

我可以帮你准备：
1. 制作过程图
2. AI概念图到成品的对比图
3. 一段可以直接发的小红书文案

这样你发起来会更完整，也能记录这个作品从想法到实物的过程。
```

## 给客户的小红书文案模板

```text
终于收到我的定制小摆件了。

一开始我只有一个很模糊的想法：【客户想法】。
没想到真的可以从AI概念图、3D模型，一步步做成实体。

最喜欢的是【细节】。
收到的时候还有展示盒，摆在桌上刚刚好。

这种感觉和直接买现成品很不一样，像是把自己脑子里的画面拿了出来。

#3D打印定制 #原创摆件 #生日礼物 #手作礼物 #桌面摆件
```
