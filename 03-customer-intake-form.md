# 客户需求表和沟通模板

## 需求表字段

```text
客户昵称
联系方式
下单渠道：小红书/抖音/朋友推荐/其他
定制用途：生日/宠物纪念/情侣纪念/原创IP/桌面摆件/其他
想做什么：开放填写
参考图上传：可多图
喜欢的风格：Q版可爱/潮玩/宠物拟人/机甲/治愈/写实/其他
目标尺寸：5cm/8cm/10cm/15cm/不确定
预算区间：100-300/300-600/600-1000/1000+
想要的交付形式：整件白膜+颜料/拆件白膜+颜料/上色成品/不确定
是否需要展示盒：是/否/不确定
期望收到日期
是否能接受AI辅助设计：是/否
是否允许成品匿名展示：是/否
备注
```

## 收到需求后的第一条回复

```text
我看完你的需求了，先帮你整理一下：

你想做的是：【一句话总结】
用途是：【用途】
风格偏向：【风格】
大概尺寸：【尺寸】
预算区间：【预算】

我建议先做【1-2个风格方向】的概念图，确认整体感觉后再评估打印和上色方案。
```

## 需求不清楚时的追问

```text
这个方向可以做，但我还需要确认3个点：

1. 你更希望它偏可爱、酷、治愈还是写实？
2. 是想做成可以直接摆放的完整成品，还是白膜自己上色？
3. 你最在意的是像参考图、礼物氛围，还是细节复杂度？
```

## 客户预算低时的回复

```text
这个预算可以做，但需要控制复杂度。

我建议优先考虑：
1. 尺寸控制在【尺寸】
2. 结构做得更整体，少拆件
3. 上色改成你自己上色，或者只做基础色块

这样能把成本压下来，也更适合第一次尝试定制。
```

## 客户需求过复杂时的回复

```text
这个想法很有画面感，但如果全部做出来，建模、拆件和上色成本会比较高。

我建议先保留最有识别度的3个元素：
1. 【核心元素1】
2. 【核心元素2】
3. 【核心元素3】

其他细节可以弱化成装饰，不然最终价格和交付周期都会上去。
```
