# 风格库和AI提示词模板

## 五种基础风格

### A. Q版可爱风

适合：生日礼物、情侣礼物、宠物、轻量预算。

关键词：

```text
Q版、圆润比例、大头小身体、表情可爱、低复杂度、适合3D打印、柔和配色
```

### B. 潮玩手办风

适合：原创IP、桌面摆件、收藏向客户。

关键词：

```text
潮玩、设计感、夸张轮廓、干净分色、玩具感、收藏摆件、适合拆件上色
```

### C. 宠物拟人风

适合：宠物纪念、宠物生日礼物。

关键词：

```text
宠物特征、拟人化、温暖、纪念感、可爱但保留真实特征、适合桌面摆放
```

### D. 机甲/游戏风

适合：男性礼物、游戏爱好者、酷感摆件。

关键词：

```text
机甲、装甲、硬边结构、模块化、战斗姿态、适合拆件打印、细节可控
```

### E. 治愈展示盒风

适合：礼物、纪念场景、展示盒套装。

关键词：

```text
治愈、微缩场景、展示盒、柔和灯光、故事感、礼物感、桌面陈列
```

## 文生概念图Prompt

```text
为一个3D打印定制摆件设计概念效果图。

主题：【主题】
用途：【生日礼物/宠物纪念/情侣礼物/原创IP/桌面摆件】
主体：【主体描述】
风格：【选择风格】
必须包含：【元素1、元素2、元素3】
避免：【不能出现的内容】
目标感觉：【可爱/治愈/酷/高级/温暖/搞怪】

要求：
1. 单个主体清晰，不要复杂背景
2. 轮廓明确，适合后续3D建模
3. 结构适合3D打印，不要过多悬空细丝
4. 颜色分区清楚，方便后续上色
5. 正面展示，干净背景，产品设计稿风格
```

## 图生图Prompt

```text
参考上传图片，为3D打印定制摆件生成概念效果图。

保留：
【需要保留的特征】

调整：
【需要夸张/可爱化/简化的部分】

风格：
【Q版可爱/潮玩/宠物拟人/机甲/治愈展示盒】

要求：
1. 保留原图最有识别度的特征
2. 转化为适合3D打印的实体摆件
3. 结构稳定，四肢和配件不要太细
4. 色块清晰，方便上色
5. 输出像产品概念图，不要像照片合成
```

## 四视图Prompt

```text
根据这个角色/摆件设计，生成用于3D建模参考的四视图。

内容：
前视图、后视图、左视图、右视图。

要求：
1. 同一个角色，同一套服装和细节
2. 四个角度比例一致
3. 白色或浅灰背景
4. 不要透视夸张，不要复杂光影
5. 轮廓、配件、颜色分区清晰
6. 适合后续图生3D模型
```

## 3D模型生成说明

```text
使用四视图生成3D模型。

目标：
适合3D打印的实体摆件模型。

要求：
1. 保持主体大轮廓和关键特征
2. 简化过细、过薄、过尖的结构
3. 让模型能稳定站立或预留底座
4. 表面不要出现破碎、孔洞、重叠结构
5. 输出适合后续修模和切片的模型文件
```
