# 3D打印作品展示站

这是一个可直接打开的静态网站原型，用于展示定制 3D 打印作品，并给客户提供三档预估报价。

## 使用方式

直接用浏览器打开：

```text
/Users/tt/codex/3d-print-order-workflow/showcase-site/index.html
```

## 可替换内容

- `index.html`：修改工坊名称、联系方式、作品标题和说明。
- `assets/`：把示例 SVG 换成真实作品照片，保持文件名或同步修改图片路径。
- `app.js`：调整报价公式、定金比例、周期和复杂度加成。
- `styles.css`：调整品牌色和布局细节。

## 当前报价方案

- 方案 A：整件白膜 + 颜料包
- 方案 B：拆件白膜 + 颜料包
- 方案 C：上色成品 + 展示盒
