# 3D打印原创定制订单工作流

这套资料用于把你的业务从“每单靠聊天推进”变成“内容引流、表单采集、模板沟通、标准报价、按状态交付”的半自动流程。

## 推荐工具组合

- 社媒发布：小红书、抖音、视频号
- 私域承接：QQ群或微信群
- 需求采集：腾讯问卷、飞书表单、金数据、Notion 表单
- 订单管理：飞书多维表格、Notion、Airtable、腾讯文档表格
- AI效果图：Midjourney、即梦、通义万相、豆包、Stable Diffusion 等
- AI 3D模型：Meshy、Tripo、Rodin 或同类图生3D工具
- 模型处理：Blender、Bambu Studio
- 交付归档：本地订单文件夹 + 云盘备份

## 总流程

```text
社媒内容发布
→ 评论/私信引导加群
→ 群公告展示案例、价格区间、需求表
→ 客户提交需求
→ 生成设计brief和初步效果图
→ 客户确认风格
→ 生成四视图和3D模型
→ 评估打印难度并报价
→ 客户支付定金
→ 建订单卡片和交付日期
→ 模型修整、拆件、切片
→ 打印、上色、组装、包装
→ 发货
→ 引导返图、好评、小红书笔记
→ 归档为下一次宣传素材
```

## 文件说明

- `01-social-media-playbook.md`：社媒内容策略、发布结构、标题文案模板
- `02-community-onboarding.md`：QQ群/微信群承接、群公告、私信自动回复
- `03-customer-intake-form.md`：客户需求表字段和沟通追问模板
- `04-style-and-prompt-templates.md`：风格模板、文生图、图生图、四视图、3D模型提示词
- `05-pricing-and-quote.md`：三档报价、成本公式、报价话术
- `06-order-tracker-template.csv`：订单状态表字段，可导入表格工具
- `07-production-sop.md`：模型检查、拆件、Bambu Studio、打印和品控流程
- `08-delivery-review-sop.md`：发货、售后、返图、好评和案例复盘
- `09-first-week-launch-plan.md`：第一周启动清单
- `10-pricing-calculator-template.csv`：报价计算表模板

## 你的人工把关点

- 客户是否是真需求
- 设计风格是否有付费价值
- 模型是否可打印
- 拆件结构是否合理
- 上色质感和最终品控
- 是否适合作为公开案例
