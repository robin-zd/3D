# 第一周启动清单

目标：用一周时间搭出最小可运行版本，不追求完美，先让客户能进群、能填表、你能报价、能推进订单。

## 第1天：搭建承接入口

完成：

```text
建QQ群或微信群
复制群公告
创建需求表
创建订单状态表
准备3-5张案例/样板图
```

当天产出：

```text
群二维码
需求表链接
订单表链接
群公告
```

## 第2天：准备样板项目

建议先做5个样板方向：

```text
宠物纪念摆件
情侣生日礼物
原创IP小人
机甲/游戏风摆件
治愈展示盒套装
```

每个样板至少准备：

```text
1张概念图
1张四视图或模型截图
1段介绍文案
1个价格区间
```

## 第3天：发布第一批内容

发布：

```text
小红书：2篇图文
抖音/视频号：1条短视频
群内：案例合集和定制说明
```

评论引导统一用：

```text
想看定制流程可以评论【定制】，我发你案例和需求表。
```

## 第4天：跑通客户沟通

模拟一个客户，从需求表到报价完整跑一遍：

```text
填表
整理brief
生成概念图Prompt
出三档报价
写定金确认话术
建订单卡片
```

## 第5天：准备报价边界

定好你的最低接单标准：

```text
最低客单价：
最低定金：
免费修改次数：
普通交付周期：
加急费用：
哪些需求不接：
哪些IP/版权风险不接：
```

## 第6天：发布第二批内容

发布内容方向：

```text
一条案例
一条避坑科普
一条“本周开放定制名额”
```

## 第7天：复盘

记录：

```text
发布了几条内容
每条浏览/点赞/收藏/评论
多少人私信
多少人进群
多少人填表
多少人询价
多少人愿意付定金
卡在哪一步
```

下周只优化一个最卡的环节。
